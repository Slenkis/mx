1 Любой том NTFS содержит загрузочный сектор, который начинается с нулевого сектора и занимает до 16 секторов
2 Базовый диск Windows Server может содержать до 4 основных или до 3 основных разделов и один дополнительный раздел
3 При преобразовании динамического диска в базовый все данные будут потеряны
4 Windows 2000 (2003) работает с файловыми системами NTFS, FAT16 и FAT32
5 Основной раздел диска —это раздел, используемый для запуска компьютера. Один из основных разделов должен быть помечен как активный
6 Разрешения NTFS можно назначить не только для папок, но и для файлов
7 В Windows 2000 (2003) отказоустойчивыми являются тома: RAID-1 и RAID-5.
8 Дополнительный раздел диска —это раздел, который может быть создан из оставшейся части свободного места.
9 Информация о дисках и основные задачи их обслуживания, например, создание и удаление разделов и томов, включены в оснастку Disk Management
10 При форматировании тома под NTFS атрибуты файла записываются в выделенное в таблице MFT пространство
11 При настройке свободного пространства на жестком диске или жесткого диска, предназначенного для Windows 2000, надо выполнить работы в следующей последовательности инициализировать диск;создать разделы или тома;отформатировать диск.
12 При перемещении файлов и папок в рамках одного раздела NTFS права доступа сохраняются
13 Чтобы описать расположение всех записей атрибутов файла, NTFS создает атрибут Attribute List
14 Как и в предыдущих версиях, в Windows 2003 размер раздела FAT16 ограничен 4 Гб
15 Вновь установленный диск в Windows Server инициализируется как базовый
16 При копировании файлов и папок в рамках одного раздела NTFS, копировании или перемещении файлов и папок на другой раздел NTFS права доступа наследуются от папки назначения
17 Windows Server поддерживает два типа дисков; базовый и динамический.
18 FAT32 поддерживает разделы до 2047 Гб.
19 Кластер — это наименьшая единица, которую ОС использует для размещения файла в разделе FAT
20 NTFS использует для описания файлов записи таблицы MFT (Master File Table)
21 Системный раздел диска —это активный раздел, содержащий файлы для загрузки ОС.
22 Простои том (simple volume) — это дисковое пространство на одном жестком диске. Простой том может занимать несколько областей (до 32) на одном диске
23 Лучшие скоростные характеристики среди всех подходов управления дисками в имеют чередующиеся тома
24 Сектор — это наименьшая единица, используемая при чтении/записи данных.
25 Раздел диска — это изолированная секция для хранения данных. Разделы позволяют хранить различные типы информации раздельно, например, пользовательские данные на одном и приложения на другом разделе.
26 Отформатированный под NTFS составной том допускает расширение за счет свободного места.
27 Зеркальныи том (mirrored volume) — это том, который состоит из двух одинаковых копий простого тома, каждая из которых находится на отдельном жестком диске.
28 Динамический диск поддерживается только в Windows 2000 (2003), делится на тома, включающие области на одном или нескольких физических дисках.. Диск может содержать простые, составные, чередующиеся (RA1D-0), зеркальные (RAID-1) и чередующиеся с четностью (RAID-5) тома
29 Нельзя расширить том, отформатированный под FAT16 или FAT32
30 Составной том (spanned volume) — это том, который включает в себя пространство нескольких дисков (до 32).При записи данных на такой том полностью заполняет сначала первый диск, затем второй и делает то же со всеми дисками в томе.
31 NTFS рассматривает каждый файл или каталог как набор атрибутов
32 Чередующийся том (striped volume) — это том, который объединяет области свободного пространства нескольких дисков (до 32) в один логический том. При работе с таким томом Windows оптимизирует выполнение, записывая данные на диски равномерно.
33 Пользователю, просматривающему общую папку под управлением DFS, не надо знать, имя сервера, на котором она находится
34 Windows 2000 (2003) использовать совместно с программами сжатия или разбиения на разделы, которым требуется доступ к дискам средствами MS-DOS нельзя
35 Загрузочный раздел (boot partition) — это основной раздел или логический диск, на котором установлены файлы ОС.
36 Базовый диск это промышленный стандарт. Такой диск делится на разделы (partition) — части, функционирующие как физически отдельные устройства хранения данных
37 При форматировании тома под NTFS, группе Everyone предоставляется для корня этого тома разрешение Full Control
38 Базовый диск Windows Server может содержать основные и дополнительные разделы и логические диски
39 При копировании файлов на тома FAT права доступа NTFS теряются


Админисрирование Microsoft Windows Server

№ Вопрос Ответ
1 Преднастроенные консоли ММС для обеспечения выполнения соответствующего набора административных задач содержат только одну оснастку
2 Сохраненная в авторском режиме консоль ММС позволяет: добавлять или удалять оснастки; создавать новые окна; видеть все части дерева консоли
3 Консоль – это набор оснасток
4 Одним из основных средств управления Windows Server,обеспечивающих стандартный способ создания, сохранения и открытия средств администрирования с помощью встраиваемых управляющих приложений, является консоль ММС (Microsoft Management Console)
5 По умолчанию Windows сохраняет файлы пользовательских консолей в папке My Administrative Tools
6 По умолчанию все новые ММС сохраняются в ______________ режиме. авторском
7 Оснастки бывают двух типов: изолированные и расширения.
8 Консоль доступна в двух режимах: авторском и пользовательском
9 Оснастка — это единица функций управления и минимальная единица расширения консоли.
10 Расширение оснастки используется для обеспечения дополнительных администра-тивных функциональ-ных возможностей для других оснасток.
11 Консоли ММС бывают двух типов: пользовательские и преднастроенные
12 Изолированная оснастка используется для администрирования Windows.
13 Пользовательские режимы консоли ММС могут быть трех типов: Full Access (Полный доступ); Limited Access, Multiple Windows (Ограниченный доступ, многооконный); Limited Access, Single Window (Ограниченный доступ, однооконный)
14 Локальная учетная запись пользователя создается в в БД безопасности данного компьютера. Windows не реплицирует информацию о локальной учетной записи на контроллеры домена
15 Помимо папки My Documents, Windows Server позволяет создать дополнительную — домашнюю — папку пользователя. Вы можете хранить домашнюю папку на компьютере клиента или в обшей папке на файловом сервере
16 Чтобы сделать профиль обязательным, переименуйте файл Ntuser.dat, расположенный на сервере, в Ntuser.man
17 Самый простой способ изменить Ваш профиль пользователя — изменить параметры рабочего стола
18 Windows Server поддерживает два типа учетной записи пользователя: доменную и локальную
19 Шаблон перемещаемого профиля копируется на общий сетевой ресурс с помощью приложения System (Система) из Control Panel.
20 Для паролей пользователей Windows Server существуют следую-щие правила пароль может содержать до 128 символов, минимальная рекомендуемая длина — 8 символов
21 Профиль пользователя – это набор папок и данных, определяющих параметры Вашего рабочего стола, приложений и место хранения личных данных.
22 Для создания доменных учетных записей служит оснастка Active Directory Users And Computers
23 Доменная учетная запись пользователя создается в организационном подразделении (ОП) в реплике хранилища Active Directory на контроллере домена.
24 Различают следующие типы групповых политик: Конфигурация программ, сценарии, службы удаленной установки, параметры безопасности, Административные шаблоны, перенаправление папок
25 Характеристиками универсальной группа домена являются: Ограниченное членство — можно добавлять членов лишь из того домена, где создана группа; Доступ к ресурсам любого домена — позволяет назначать разрешения доступа к ресурсам в любом домене; Доступна лишь в доменах смешанного режима
26 Для создания групп служит оснастка Active Directory Users And Computers.
27 Изолированные локальные группы позволяет создать оснастка Computer Management
28 Характеристиками глобальной группы домена являются Ограниченное членство — можно добавлять членов лишь из того домена, где создана группа; Доступ к ресурсам любого домена — позволяет назначать разрешения доступа к ресурсам любого домена.
29 В Windows определено четыре типа встроенных групп глобальные, доменные локальные, изолированные локальные и си-стемные.
30 В Windows Server существуют два типа групп безопасности и распространения
31 Характеристиками локальной группы домена являются Открытое членство — можно добавлять членов из любого домена; Доступ к ресурсам одного домена — позволяет назначать разрешения доступа к ресурсам того же домена, где была создана группа.
32 Группа Everyone (Все) является встроенной системной
33 В качестве списков пользователей для осуществления функций, не связанных с системой защиты, приложения используют группы распространения
34 Группа (group) — это набор учетных записей пользователей.
35 В ОС Windows доступны только группы безопасности
36 Групповые политики (group policies) — это наборы параметров конфигурации, применяемых к одному или нескольким объектам в хранилище Active Directory.
